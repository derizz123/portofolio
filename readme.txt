"empty-site" adalah kumpulan2 folder kosong yang saya susun sesusai kaidah yang biasa di terapkan pada FRAMEWORK (tetapi ini hanya empty folder), yang gunanya untuk
memudahkan seorang Programmer Web dalam pembuatan folder secara manual, sehingga dapat mempercepat pekerjaan tanpa creating folder terlebih dahulu.

Direktori file
-asset
  -css			= folder ini hanya untuk style css aja tidak boleh ada file lain.
  -js			= folder ini hanya untuk file .js yang file js nya sendiri berupa function (untuk memudahkan dalam pengembangan), dan isi dari folder ini hanya untuk jquery.js, json dan file .js yang anda buat saja. 
  -extension		= extension ini untuk all plugin yang didalamnya harus ada folder (untuk setiap 1 struktur plugin) nama plugin/extension itu sendiri
  -images		= folder ini hanya untuk file yang bertipe image/gambar

-pages			= adalah file yang bertujuan sebagai antar-muka pada user 
-model			= adalah file yang bertujuan sebagai yang bertindak/berinteraksi dengan database MDL
-controller		= adalah file yang bertujuan untuk mengontrol file 'pages' dan model pada pengolahan (Insert Edit Delete) dan pengiriman (SELECT) data
